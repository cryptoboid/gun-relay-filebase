- AWS_SECRET_ACCESS_KEY ([see here](https://gun.eco/docs/Using-Amazon-S3-for-Storage))
- NPM_CONFIG_PRODUCTION ([see here](https://gun.eco/docs/Using-Amazon-S3-for-Storage))
